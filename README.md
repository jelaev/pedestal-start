  <code>git clone https://github.com/jelaev/pedestal-start.git [app-name]</code></p>

  <code>clj -M:nrepl</code></p>

<p>If you want onl run to see
  <code>clj -M:prod -m app.main</code>
  go to
  <code>http://localhost:8030/health</code></p>
